# Back

## Development server

Para rodar a aplicação em perfil de desenvolvimento basta rodar:

```bash
mvn spring-boot:run
```

A API estará rodando no seguinte endereço http://localhost:8080

## Building

Caso queira ser feito um build, é possível por meio do seguinte comando:

```bash
mvn package
```

## Running unit tests

Para rodar todos os testes em um ambiente controlado:

```bash
mvn test
```
